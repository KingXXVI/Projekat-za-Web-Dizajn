First Update

by Uros Milosevic
published March 15th, 2021

I probbably should've done this first update a lot earlier but i got carried playing around with css and getting the site to look decent. The color scheme was taken from the wallpaper i use for my desktop PC and phone although i think it might be a tad too dark.The site is prety barebones at the moment only having 3 pages and a bit of css to make itnot look prehistoric. Though i have gone for a darker theme which is something that very few sites actually offer by default. As for the light theme i don't think i will be adding one, partly beacuse i dont like light themes but also bacause i don't want to have to create a color scheme for it. That being said knowing me i will probbably end up making a light theme when i get bored.

In any case i have nothing more to say right now so i will put the color scheme for the site here along with some filler text to showcase the page scrolling

background color - #323232
text background color - #424242
shadow color - #222
navigatio bar background - #656565
highlight color - #CCA421
home and about button background - #999999
text color - #cfcfcf
highlight text color - #252525
